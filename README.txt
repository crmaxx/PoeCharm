Предупреждение:
* Полная оригинальная версия вставляется в переводчик без вмешательства в исходную модель расчета для поддержки русского языка. Пожалуйста, по всем ошибкам обращайтесь на GITHUB оригинального автора.
* По вопросам руссификации обращайтесь ко мне.
* Локализованная версия не поддерживает вставку символов в национальной кодировке.
* Пожалуйста, используйте английский язык для поиска предметов экипировки и нодов Дерева талантов локализованной версии сервера,  русский язык не поддерживается

FAQ:

>> POB напоминает об обновлении, можно ли нажимать?
* Быстрое обновление в POB относится к собственному обновлению POB, нажмите «Обновить» = «Обновить POB».

>> Как поместить оригинальный архив POB BD в PoeCharm?
1) Откройте каталог builds в исходном каталоге POB.
2) Щелкните правой кнопкой мыши значок запуска PoeCharm Pob и выберите «Открыть архив».
3) Скопируйте

>> Можно ли изменить шрифт?
* Вы можете заменить файл шрифта в каталоге программы PoeCharm/Fonts, замените FZ_ZY.ttf

>> 国服(国际服)链接导入国际服(国服)只有天赋没有装备?
* Архивы национальных и международных версий серверов несовместимы, не используйте их вместе.

>> Как добавить другую версию POB в PoeCharm?
* Скопируйте весь каталог POB в основной каталог программы PoeCharm, перезапустите PoeCharm
